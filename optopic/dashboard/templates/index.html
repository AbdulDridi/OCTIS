Welcome to OpTopic!

What is OpTopic?

OpTopic is an open-source evaluation and optimization framework that allows you to optimize the hyperparameters of state-of-the-art Topic Models and compare their performance with respect to several evaluation metrics on several datasets.

Main Features

You can (1) define your experiments directly from this dashboard (link ..) and (2) compare their optimization progress ( link...) and (3) inspect the output of each topic model (link...)
